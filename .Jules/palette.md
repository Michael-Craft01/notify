## 2025-02-28 - Missing ARIA Labels on Icon-only Buttons and State Attributes
**Learning:** Found a recurring accessibility issue where icon-only interactive elements (like Edit, Trash, Close, Search Clear buttons) are missing `aria-label`s, which is critical for screen reader support. Furthermore, dynamic state attributes like `aria-expanded` on toggles and `aria-pressed` on selected view mode buttons are consistently omitted across components like `AssignmentCard`, `AssignmentContainer`, and `SettingsModal`.
**Action:** When adding or reviewing interactive icon-only buttons or state-based toggles in this codebase, explicitly ensure an `aria-label` is provided and standard ARIA state attributes (`aria-expanded`, `aria-pressed`) are accurately bound to their corresponding React states.
